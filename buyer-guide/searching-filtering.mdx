Alongside general browsing, Annapurna Art supports narrowing down artwork through search and filters. This page describes how these tools generally work and what's confirmed about them.

<Info>
  Annapurna Art's category pages support a filter parameter (for example, a painting category page can be viewed with a filter applied), which confirms that filtering exists in some form. The exact filter options, labels, and search behavior haven't been independently verified against the live product.
</Info>

## Searching for artwork

If you're looking for something specific, a search tool is the fastest way to find it. On most marketplaces, a search bar lets you enter keywords, such as an artist's name, a medium, or a subject, and returns matching artwork.

<Steps>
  <Step title="Open the search bar">
    Look for a search icon or input field, typically in the site header or navigation menu.
  </Step>
  <Step title="Enter your search terms">
    Type a keyword related to what you're looking for, such as an artist name, artwork title, medium, or theme.
  </Step>
  <Step title="Review the results">
    Browse the matching artwork. Results likely show the same thumbnail, title, artist, and price information you'd see when browsing a category.
  </Step>
  <Step title="Refine with filters, if available">
    Apply any available filters to narrow the results further.
  </Step>
</Steps>

## Filtering within a category

Category pages on Annapurna Art support a filter option, confirmed by the site's URL structure when viewing a category. This suggests you can narrow a category, such as painting, down to a more specific subset.

<Warning>
  The exact filter criteria (such as price range, medium, size, or availability) have not been independently verified. Check the filter controls shown on the live category page to see what's currently offered.
</Warning>

## Typical filter categories on art marketplaces

As with most art marketplaces, filters are commonly organized around a few dimensions. The table below outlines common filter types you might expect to see, alongside their verification status on Annapurna Art.

| Filter type | Description | Confirmed on Annapurna Art? |
|---|---|---|
| Category | Medium such as painting, sculpture, photography, or drawing | Yes — confirmed via category pages |
| Theme or country | Artwork grouped by region or subject | Yes — confirmed via theme/country browsing |
| Price range | Minimum and maximum price | Not verified |
| Size or dimensions | Physical size of the piece | Not verified |
| Availability | Whether a piece is still for sale | Not verified |

<Tip>
  If you're not sure whether a filter you need exists, browsing by category or theme first and then scanning results manually is a reliable fallback. See [Browsing Artwork](/buyer-guide/browsing-artwork) for more on that approach.
</Tip>
